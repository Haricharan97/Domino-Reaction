# Domino-Chain-Reaction 
A domino simulator
Creates a satisfying domino setups, trigger realistic chain reactions, and explore momentum-based physics inside an interactive 3D environment.<br>
<br>Features:
<br>3D domino simulation
<br>Simple building system
<br>Camera controls
<br>Realistic chain reactions
<br>Reset and clear options
<br>WebGL rendering with Three.js
